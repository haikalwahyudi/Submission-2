About

Aplikasi ini adalah sebuah aplikasi berbasis web dengan menerapkan teknologi PWA(Progresive Web Apps) dengan menggunakan teknologi PWA web yang kita buat bisa diakses secara offline dan layaknya aplikasi native seperti aplikasi android, kita cukup membuat satu aplikasi dan bisa dijalankan di dua tempat android dan web, Dalam aplikasi ini memuat informasi terkait club club pemain sepak bola beserta jadwal tandinnya, kita bisa melihat club kesayangan kita dan juga melihat jadwal tandingnya melalui aplikasi ini, Saya membuat aplikasi web ini untuk menyelesaikan Submission 2 dari Dicoding academy dan semoga bisa lolos.
